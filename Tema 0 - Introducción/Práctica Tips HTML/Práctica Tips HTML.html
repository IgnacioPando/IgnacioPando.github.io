7. Window.opener

Las páginas abiertas con target="_blank" permiten que la nueva página acceda al window.opener original. Esto puede tener implicaciones de seguridad y rendimiento. Incluir rel="noopener" o rel="noreferrer" para prevenir esto.

9. Eliminación de caché de favicon

Para actualizar el favicon de su sitio web, puede obligar a los navegadores a descargar una nueva versión agregando al nombre del archivo ?v=2 .
Esto es especialmente útil en producción para asegurarse de que los usuarios obtengan la nueva versión.
